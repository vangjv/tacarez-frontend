## TacarEZ

## Introduction

This application provides a user experience to help map data using ArcGIS services.  There is also an approval workflow built in that lets users save changes and send them to stakeholders for approval.

## Technologies

This application is built on Angular and has .NET Azure functions built in. 

## Installation

This application can be cloned via this git repository.  

`git clone https://github.com/vangjv/tacarez-frontend/`
